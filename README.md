# 🍽️ Luxury Restaurant Website (Frontend Project)

A modern, responsive restaurant website built with HTML, Tailwind CSS, and vanilla JavaScript.  
Designed as a high-end UI/UX concept for a real-world restaurant business.

---

## ✨ Features

- Fully responsive design (mobile + desktop)
- Luxury UI styling with custom branding system
- Dynamic menu rendering using JavaScript
- Image-based food menu layout
- Interactive modal popups for menu items
- Floating action buttons (View Menu / Order Now)
- Social media & contact integration
- Clean, production-style structure

---

## 🧠 Tech Stack

- HTML5
- Tailwind CSS
- JavaScript (Vanilla)
- Font Awesome Icons
- Google Fonts (Playfair Display + Inter)
